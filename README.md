# 🚀 QuizForge – Full-Stack Tech Interview Preparation Platform

🔗 **Live Demo:** https://quizforge-app-bddn.onrender.com/
🔗 **GitHub Repo:** https://github.com/Bharathi0608/Quizforge

---

## 📌 Overview

**QuizForge** is a full-stack web application built to help users prepare for technical interviews through interactive quizzes.

It offers a smooth and modern user experience with authentication, score tracking, and leaderboard functionality — making learning both engaging and competitive.

---

## ✨ Features

* 🔐 User Authentication (Signup & Login)
* 🧠 Interactive Quizzes (Multiple Categories)
* 📊 Real-time Score Tracking
* 🏆 Leaderboard System
* ⚡ Smooth Animations (Framer Motion)
* 🌐 Fully Responsive UI
* 🚀 Deployed on Render (Frontend + Backend)

---

## 🛠️ Tech Stack

### 🎨 Frontend

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Bharathi0608/Quizforge.git
cd Quizforge
```

---

### 2️⃣ Install Dependencies

#### Frontend

```bash
npm install
```

#### Backend

```bash
cd backend
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env` file inside the **backend folder**:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

---

### 4️⃣ Run the Project Locally

#### Start Backend

```bash
cd backend
npm start
```

#### Start Frontend

```bash
npm run dev
```
